---

## 🔹 Modules
- Click GUI with toggle states  
- Velocity limiter + glide  
- Tracer line draw  
- Lag-safe script injection

## 🛠️ How to Use

1. Run `Loader.exe`  
2. Attach to Minecraft  
3. GUI loads by default
